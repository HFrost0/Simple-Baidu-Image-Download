# Simple-Baidu-Image-Download
只有30行的百度图片爬虫，只用最简单的语句，适合爬虫初学者
## 环境
需要安装requests, tqdm
```
pip install requests, tqdm
```
## 运行
```
python baidu_image.py
```
然后按照提示输入关键词和想要下载的数量
